# Kron

Gestor de actividades, tareas y colaboradores para equipos de trabajo.

## Descripción
Kron es una aplicación web desarrollada en PHP para la gestión de actividades, tareas y equipos. Permite clonar actividades, gestionar tareas, asignar colaboradores y eliminar actividades de forma segura y eficiente.

## Características principales
- Clonación de actividades con tareas y colaboradores
- Eliminación de actividades y datos relacionados
- Gestión de roles, equipos y usuarios
- Interfaz intuitiva con tarjetas resumen y acciones rápidas

## Instalación
1. Clona el repositorio:
   ```
   git clone https://github.com/NewTekspa/Kron.git
   ```
2. Accede a la carpeta del proyecto:
   ```
   cd Kron
   ```
3. Configura la base de datos en `config/database.php`.
4. Inicia el servidor local:
   ```
   php -S localhost:8000 -t public
   ```

## Uso
- Accede a la aplicación desde tu navegador en `http://localhost:8000`.
- Gestiona actividades, tareas y colaboradores desde el panel principal.

## Estructura del proyecto
- `app/Controllers`: Controladores de la lógica de negocio
- `app/Models`: Modelos de datos
- `app/Views`: Vistas y plantillas
- `config/`: Configuración de la aplicación y rutas
- `public/`: Archivos públicos y punto de entrada
